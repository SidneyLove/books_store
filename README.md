Book Store — это веб-приложение на React, реализующее функционал интернет-магазина книг. Проект демонстрирует навыки работы с компонентами, состоянием приложения и пользовательским интерфейсом.

Основные возможности

Просмотр списка доступных книг

Добавление и удаление товаров из корзины

Категории для удобной навигации

Оформление заказа с формой регистрации

Кастомная стилизация и подключение собственных шрифтов

Стек технологий

React — библиотека для создания интерфейсов

CSS — для оформления компонентов

React Hooks (useState, useEffect) — управление состоянием

SVG и кастомные шрифты — для улучшения UI
